# RedlineRift

RedlineRift is a realistic racing game built with **HTML, CSS, and JavaScript**. It focuses on smooth controls, sharp handling, and a clean UI—rewarding precise braking, clean lines, and consistent lap times.

## Features
- Responsive keyboard controls
- Realistic-feeling handling (grip, drift, braking)
- Lap time / speed HUD
- Clean, simple UI

## Tech Stack
- HTML
- CSS
- JavaScript
